# XpressDM

This project is a frontend assignment given to me by XpressDM. I was provided with a figma link and asked to code as close as possible using Next JS.

## Summary

XpressDM is a company that offers a range of services. As part of an assignment, I was tasked with creating a responsive pricing page that showcases multiple plans and features. The project also includes a comparison section where users can compare the different plans and features.

## Features

- Responsive pricing page with multiple plans
- Comparison section to compare plans and features
- Built using Next JS

## Technologies Used

- Next JS
- Tailwind CSS
- TypeScript

## How to Run

1. Clone the repository
2. Run `npm install` to install all dependencies
3. Run `npm run dev` to start the development server
4. Open `http://localhost:3000` in your web browser to view the project
